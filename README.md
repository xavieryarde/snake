# How to Play:
Keyboard:
W / Up - Move up

S / Down - Move down

A / Left - Move Left

D / Right - Move Right



Controller:
D-Pad / Left Stick to move snake

A (Xbox) / X (PS) - When on Game Over Screen to Play Again


Enjoy :)
